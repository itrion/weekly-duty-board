## Packages
date-fns | Date manipulation for weekly calendar logic
clsx | Utility for conditional class names (usually installed but good to ensure)
tailwind-merge | Utility for merging tailwind classes (usually installed but good to ensure)

## Notes
The app requires a print layout.
@media print styles are critical for hiding UI elements and formatting the table for A4 landscape.
Lucide icons are used for task visualization.
Theme is "Petrol Green" (#006D77).
